# Bergamota

Personal portfolio site for **Marcelo Barella**, branded as **Bergamota**. Single-page Nuxt 4 app: hero, projects, blog index, and contact.

## Requirements

- Node.js (LTS recommended)
- npm

## Commands

| Script        | Action                    |
| ------------- | ------------------------- |
| `npm install` | Install dependencies      |
| `npm run dev` | Dev server (port 3027)    |
| `npm run build` | Production build        |
| `npm run preview` | Serve production build locally |

Dev server binds to `0.0.0.0:3027` per `nuxt.config.ts`.

## Stack

Nuxt 4, Vue 3, Nuxt UI, Tailwind CSS 4, GSAP / nuxt-anime.
